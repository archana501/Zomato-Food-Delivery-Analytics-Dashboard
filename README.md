# Zomato-Food-Delivery-Analytics-Dashboard

## Project Overview

This project presents an interactive **Zomato Food Delivery Analytics Dashboard** built using **Tableau Public**. The dashboard analyzes revenue, operations, customer behavior, and restaurant performance to provide meaningful business insights.

The dashboard is divided into four interactive pages, each focusing on a different aspect of the business.

---

## Tools Used

- Tableau Public
- Microsoft Excel
- Power Query

---

## Dashboard Pages

### Executive Overview

**KPIs**
- Total Orders
- Total Revenue
- Net Revenue
- Average Rating
- Average Delivery Time

**Visualizations**
- Revenue by City
- Revenue by Cuisine Type
- Revenue by Payment Method
- Monthly Revenue Trend
- Orders by Meal Time
